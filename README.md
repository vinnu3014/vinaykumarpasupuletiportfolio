## ğŸ“„ Resume

To generate a PDF resume:

### Option 1: Using Browser (Easiest)
1. Open `assets/resume.html` in your browser
2. Press `Cmd+P` (Mac) or `Ctrl+P` (Windows/Linux)
3. Select "Save as PDF"
4. Save as `Vinay_Kumar_Resume.pdf` in the `assets` folder

### Option 2: Using Command Line (if you have wkhtmltopdf installed)
```bash
# Install wkhtmltopdf (Mac)
brew install wkhtmltopdf

# Install wkhtmltopdf (Ubuntu/Debian)
sudo apt-get install wkhtmltopdf

# Generate PDF
wkhtmltopdf assets/resume.html assets/Vinay_Kumar_Resume.pdf
```

### Option 3: Online Converter
1. Open `assets/resume.html` in your browser
2. Use an online HTML to PDF converter
3. Save the PDF to the `assets` folder
