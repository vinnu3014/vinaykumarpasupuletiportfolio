# Vinay Kumar Resume - Placeholder

This is a placeholder file. To generate your actual resume PDF:

1. Open `assets/resume.html` in your web browser
2. Press Cmd+P (Mac) or Ctrl+P (Windows/Linux)
3. Select "Save as PDF" as the destination
4. Save as "Vinay_Kumar_Resume.pdf" in the assets folder

Or use the generate_resume.sh script:
./generate_resume.sh

Your resume will then be available for download from your portfolio website.
